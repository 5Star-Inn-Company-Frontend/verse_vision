# Song Management

Manage your church's song library with ease.

![Song Library Overview](/docs/screenshots/songs-library.png)
> *Please upload a screenshot of the song library list to: `public/docs/screenshots/songs-library.png`*

## Adding Songs
1. Click **Add Song**.
2. Enter the title, author, and CCLI number.
3. Type lyrics in the editor.

![Add Song Dialog](/docs/screenshots/songs-add.png)
> *Please upload a screenshot of the 'Add Song' dialog or screen to: `public/docs/screenshots/songs-add.png`*

## Editing Lyrics
- Use standard text editing.
- Label sections (e.g., [Verse 1], [Chorus]).

![Lyrics Editor](/docs/screenshots/songs-editor.png)
> *Please upload a screenshot of the lyrics editor in action to: `public/docs/screenshots/songs-editor.png`*

## Importing Songs
Import songs from CCLI SongSelect or text files.
