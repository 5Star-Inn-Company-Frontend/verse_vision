# Audio Transcription

Turn spoken words into text.

![Transcription Interface](/docs/screenshots/transcription-view.png)
> *Please upload a screenshot of the transcription or captioning view to: `public/docs/screenshots/transcription-view.png`*

## Features
- **Live Captioning**: Display captions for the hearing impaired.
- **Sermon Notes**: Generate a transcript of the message.

## Usage
Ensure your audio input is clear for the best accuracy.
