# The Interface

VerseVision's interface is designed to be intuitive and powerful.

## Main Dashboard
The dashboard provides an overview of your recent activity and quick access to key features.

![Main Dashboard Screenshot](/docs/screenshots/interface-dashboard.png)
> *Please upload a screenshot of the main dashboard to: `public/docs/screenshots/interface-dashboard.png`*

## Sidebar Navigation
- **Songs**: Manage your lyric library.
- **Scriptures**: Search and display Bible verses.
- **Media**: Access backgrounds and videos.
- **Schedule**: Plan your service order.
- **Cameras**: Configure camera inputs.
- **Settings**: Adjust application preferences.

![Sidebar Navigation Screenshot](/docs/screenshots/interface-sidebar.png)

## Preview & Output
- **Preview Window**: See what's coming up next.
- **Live Output**: View what is currently being projected to the congregation.
