# Installation

This guide will walk you through the installation process for VerseVision.

## System Requirements

Before installing VerseVision, ensure your system meets the following requirements:
- **Operating System**: Windows 10/11, macOS, or Linux.
- **Memory**: Minimum 4GB RAM (8GB recommended).
- **Storage**: At least 500MB of free space.
- **Internet**: Required for AI features and updates.

## Step-by-Step Installation

1.  **Download the Installer**: 
    Visit the [Download Page](/) and download the latest version for your operating system.

2.  **Run the Installer**:
    - **Windows**: Double-click the `.exe` file and follow the on-screen instructions.
    - **macOS**: Open the `.dmg` file and drag VerseVision to your Applications folder.
    - **Linux**: Run the AppImage or install via your package manager.

3.  **Launch the Application**:
    Open VerseVision from your desktop or applications menu.

4.  **Sign In (Optional)**:
    Create an account or sign in to access cloud features and sync your settings.

## Post-Installation

After installation, for offline use of Scripture Detection feature, the app need to download the required models, ensure your device is connected to the internet for the first time setup untill you see Offline AI Ready
