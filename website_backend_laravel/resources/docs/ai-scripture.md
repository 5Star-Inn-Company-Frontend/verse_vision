# Scripture Detection

Never miss a verse during a sermon.

![Scripture Detection Settings](/docs/screenshots/ai-scripture-settings.png)
> *Please upload a screenshot of the Scripture Detection settings to: `public/docs/screenshots/ai-scripture-settings.png`*

## How it Works
VerseVision listens to the speaker's audio. When a scripture reference is detected (e.g., "As it says in Genesis 1:1..."), the system automatically prepares or displays the verse.

## Configuration
- **Auto-Show**: Automatically project the verse (use with caution).
- **Queue Only**: Add the detected verse to a queue for operator approval.
