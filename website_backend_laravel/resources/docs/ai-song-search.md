# AI Song Search

Find lyrics instantly without manual typing.

![AI Song Search Interface](/docs/screenshots/ai-song-search.png)
> *Please upload a screenshot of the AI Song Search tool to: `public/docs/screenshots/ai-song-search.png`*

## How to Use
1. Go to the **Songs** tab.
2. Click **AI Search**.
3. Enter the song title or a line of lyrics.
4. Select the correct match to import.
