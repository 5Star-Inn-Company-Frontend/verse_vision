# Scripture Projection

Display Bible verses instantly.

![Scripture Tab Overview](/docs/screenshots/scriptures-overview.png)
> *Please upload a screenshot of the main Scripture tab to: `public/docs/screenshots/scriptures-overview.png`*

## Selecting a Bible Version
Choose from available translations (KJV, NIV, ESV, etc.) in the dropdown menu.

## Searching for Verses
- Type a reference (e.g., "John 3:16").
- Search by keyword (e.g., "Love").

![Scripture Search Results](/docs/screenshots/scriptures-search.png)
> *Please upload a screenshot showing scripture search results to: `public/docs/screenshots/scriptures-search.png`*

## Display Options
- **Quick Show**: Immediately project the selected verse.
- **Add to Schedule**: Place the scripture in your service lineup.
