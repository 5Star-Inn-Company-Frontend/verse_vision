# Media & Backgrounds

Enhance your projection with beautiful visuals.

![Media Library Overview](/docs/screenshots/media-library.png)

## Supported Formats
- **Images**: JPG, PNG.
- **Videos**: MP4, MOV (looping supported).

## Managing Your Library
- Upload new media files.
- Organize into folders or tags.
- Set default backgrounds for songs or scriptures.
