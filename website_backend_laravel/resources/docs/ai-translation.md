# Live Translation

Translate your service in real-time for a multilingual congregation.

![Live Translation Settings](/docs/screenshots/ai-translation-settings.png)
> *Please upload a screenshot of the Live Translation configuration settings to: `public/docs/screenshots/ai-translation-settings.png`*

## Setup
1. Enable **Live Translation** in Settings.
2. Select your input language (e.g., English) and target languages (e.g., Spanish, French).

## During Service
- The AI listens to the audio input.
- Translations appear as subtitles on the screen or via the mobile app.
