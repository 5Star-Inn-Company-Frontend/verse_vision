# OBS Integration

Seamlessly connect with Open Broadcaster Software (OBS).

![OBS Settings Panel](/docs/screenshots/obs-settings.png)
> *Please upload a screenshot of the OBS integration settings panel to: `public/docs/screenshots/obs-settings.png`*

## Setup
1. Enable the **WebSocket Server** in OBS (Tools > WebSocket Server Settings).
2. In VerseVision, go to **Settings > OBS**.
3. Enter the Port and Password.
4. Click **Connect**.

## Features
- Trigger OBS scenes from VerseVision.
- Send lower-thirds lyrics directly to OBS.
