# Camera Setup

Integrate live video feeds.

![Camera Configuration](/docs/screenshots/camera-settings.png)
> *Please upload a screenshot of the camera input selection screen to: `public/docs/screenshots/camera-settings.png`*

## Supported Devices
- Webcams
- Capture Cards (Elgato, Blackmagic, etc.)
- IP Cameras
- Mobile Phones (via VerseVision Mobile App)

## Configuration
Go to **Settings > Video** to select your input device and resolution.
