VerseVision User Guide

Welcome to VerseVision, your AI-powered worship technology solution. This guide will help you understand how to use the platform effectively.

1. Getting Started

When you launch VerseVision, you are presented with the Operator Dashboard. This is your command center for managing live lyrics, scripture, and camera feeds.

  • Operator View: The main interface where you control what is displayed.
  • Program Output: The clean feed that goes to your projector or OBS.

2. Scripture Detection

AI-Powered Detection

VerseVision uses advanced AI to listen to your service audio and automatically detect scripture references.

  • Ensure your microphone is selected in the settings.
  • The system listens for phrases like "John 3:16" or "Genesis Chapter 1 Verse 1".
  • Detected scriptures will appear in the Queue.
  • Click "Show" to display a scripture on the main screen.

Manual Search

You can also manually search for scriptures using the search bar at the top right. Just type a reference (e.g., "Ps 23") and press Enter.

3. OBS & EasyWorship Integration

To send VerseVision's output to external broadcasting software:

OBS Studio

  1. Click the "Connect" button in the Program Preview toolbar.
  2. Choose Window Capture (capture the popup window) or Browser Source (paste the provided URL).
  3. Set dimensions to 1920x1080.

EasyWorship 7+

You can integrate VerseVision into EasyWorship using one of two methods:

  • Web Method (Recommended): Add a new Web media item in EasyWorship with the URL http://localhost:3000/program?bg=transparent. This provides a clean overlay with no background.
  • Green Screen (Chroma Key): Open the Program Window with a green background (available in the Connect modal). Capture this window in EasyWorship (via NDI or Feed) and enable Chroma Key to remove the green background.

4. Live Camera & Lower Thirds

VerseVision can overlay lyrics and scripture on top of a live camera feed.

  • Select your camera source from the dropdown menu in the dashboard.
  • Use the "Overlay Mode" toggle to switch between full-screen backgrounds and transparent lower-thirds.

5. Troubleshooting

  • Audio not picking up? Check your system's default microphone settings and ensure VerseVision has permission to access it.
  • Scripture not detecting? Try speaking clearer or closer to the microphone. Ensure the "Offline AI" or "Cloud AI" engine is selected correctly.
  • Window not showing? Use Ctrl+R (Cmd+R on Mac) to reload the application.
